Cadenas de Texto

Las cadenas de texto, comúnmente conocidas como strings en muchos lenguajes de programación, son una secuencia de caracteres, como letras, números, y símbolos, que se utilizan para representar texto. En la mayoría de los lenguajes de programación, las cadenas de texto se pueden crear utilizando comillas simples (''), comillas dobles ("") o triples (''' o """) dependiendo de la sintaxis específica del lenguaje.

Por ejemplo, en Python, una cadena de texto se puede definir de la siguiente manera:

cadena = "Hola, mundo!"

Enteros

Los enteros son números sin parte fraccionaria, es decir, números que no tienen decimales. Pueden ser positivos, negativos o cero.

Por ejemplo, en Python, los enteros se pueden definir de la siguiente manera:

numero_entero = 42

Los enteros son uno de los tipos de datos más comunes y fundamentales en la programación.

Decimales

Los decimales, también conocidos como números de punto flotante o floats en muchos lenguajes de programación, son números que pueden tener una parte fraccionaria.

Por ejemplo, en Python, los decimales se pueden definir de la siguiente manera:

numero_decimal = 3.14

Boolean

Los booleanos son un tipo de dato que representa un valor de verdad, es decir, verdadero o falso. En la mayoría de los lenguajes de programación, los booleanos se utilizan para realizar operaciones lógicas y de control de flujo, como la evaluación de condiciones y la toma de decisiones.

Por ejemplo, en Python, los booleanos se pueden definir de la siguiente manera:

booleano = True

Estructuras de Control: while

La estructura de control while se utiliza para repetir un bloque de código mientras una condición dada sea verdadera. En cada iteración del bucle, se evalúa la condición y, si es verdadera, se ejecuta el bloque de código.

Por ejemplo, en Python, un bucle while se puede definir de la siguiente manera:

contador = 0
				while contador < 5:
					print(contador)
					contador += 1

Estructuras de Control: for

La estructura de control for se utiliza para iterar sobre una secuencia (como una lista, tupla, rango, etc.) y ejecutar un bloque de código una vez para cada elemento de la secuencia.

Por ejemplo, en Python, un bucle for se puede definir de la siguiente manera:

for i in range(5):
					print(i)

Estructuras de Control: if

La estructura de control if se utiliza para ejecutar un bloque de código si una condición dada es verdadera. Si la condición es falsa, el bloque de código dentro del if no se ejecuta.

Por ejemplo, en Python, una declaración if se puede definir de la siguiente manera:

numero = 10
				if numero > 0:
					print("El número es positivo")

Estructuras de Control: if-else

La estructura de control if-else se utiliza para ejecutar un bloque de código si una condición dada es verdadera, y otro bloque de código si la condición es falsa.

Por ejemplo, en Python, una declaración if-else se puede definir de la siguiente manera:

numero = -5
				if numero > 0:
					print("El número es positivo")
				else:
					print("El número es negativo o cero")
