Diccionario de Datos

El diccionario de datos es una herramienta utilizada en bases de datos que por su naturaleza trabaja con muchas variables.
Es útil aprender a manejar con soltura este tipo de herramientas, pues nos ahorran tiempo y permiten comprender cualquier estructura de cualquier base de datos.
NOTA: NO todas las bases de datos van a trabajar con diccionarios de Datos, pero es probable que entre mas compleja sea, el uso de un diccionario puede ser muy útil.
