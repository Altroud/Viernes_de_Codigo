¿Qué es un dataframe?

Un DataFrame es como una tabla en una hoja de cálculo. Es una forma de organizar datos en filas y columnas, como una lista de listas donde cada fila tiene información relacionada.

Importación de dataframes: Introducción a Pandas

Pandas es un módulo de Python utilizado para manipular y analizar datos de manera eficiente. Pandas nos permite importar distintos formatos. Veámoslo con un ejemplo
