String o Cadenas de Texto

Son variables de tipo alfabético, es decir, texto plano.


            a = "Hola Mundo :D"
            b = "Esta clase es aburrida."
            c = "esta clase no es aburrida."

Floats o numeros decimales

Son variables de tipo numerica, con parte entera y decimal. 

 
                    a = 5.5
                    b = 0.33
                    c = "0.55"

Boolean o variables falso/verdadero

Son variables de tipo condicional, falso o verdadero. 

 
                    a = False
                    b = True
                    c = "True"

Int o numeros enteros

Son variables de tipo numérico, solo con parte entera. 

 
                    a = 2
                    b = 4
                    c = 5
